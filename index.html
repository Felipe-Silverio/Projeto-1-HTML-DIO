O que são tags HTML

(Voltar)

Tags HTML são elementos fundamentais da linguagem HTML (HyperText Markup Language) utilizados para criar e estruturar o conteúdo de páginas web. Cada tag HTML é um comando que define como um pedaço de conteúdo deve ser exibido no navegador. As tags são delimitadas por colchetes angulares (<>) e geralmente vêm em pares: uma tag de abertura e uma tag de fechamento, com o conteúdo a ser exibido posicionado entre elas.

Tags HTML e Boas Praticas

(Voltar)

Aqui irei listar algumas das tags que aprendi na primeira aula

  • Blockquote: Tag usada para quando a frase é um citação.
  • U: Tag para sublinhar o texto.
  • Mark: Tag para marcar o texto.
  • Sup: Tag usada para deixar a frase superescrita.
  • Sub: Tag usada para deixar a frase sobreescrita
  • Ol: Lista ordenada.
  • Ul: Lista não ordenada.
  • Li: Itens da lista.
  • Hr/: Essa tag faz uma linha em visisão a outra tag.

    Boas Praticas

  • Mantenha seus visitantes no seu site ao abrir páginas internas na mesma aba. Isso ajuda a criar uma navegação fluida e intuitiva, incentivando os usuários a explorarem mais o conteúdo do seu site.
  • Quando for necessário direcionar seus visitantes para um site externo, configure esses links para abrir em uma nova aba. Isso garante que o seu site permaneça acessível, permitindo que os usuários retornem facilmente após visualizar o conteúdo externo.
